## Run Locally

Clone the project

```bash
  git clone https://github.com/se-bayan-elian/ecommerce-app
```

Go to the project directory

```bash
  cd ecommerce-app
```

Install dependencies

```bash
  yarn 
```

Very important note : this website is not respnsive ,, please sure your scale'width is 1920px
